# CSC 591 Internet of Things - Course Project

## Installation
```
$ pip3 install Flask
$ pip3 install flask-socketio
$ pip3 install python-dotenv
$ pip3 install wiotp-sdk
```

## Environment Set Up
Create `.env` file in the current directory. Input following into `.env` file.
```
HOST=<IPv4 address>
```

## Execution
```
$ python3 main.py
```
